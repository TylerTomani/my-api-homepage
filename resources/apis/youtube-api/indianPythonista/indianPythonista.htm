By default, the search() method returns 5 items in JSON and each item has 5 keys:

  • kind
  • etag
  • regionCode
  • pageInfo
  • items
for item in response:
print(item)

The first 'kind' is #SearchListResponse, and nested within its 'items' key is another 'kind' searchResult.
The search result items are contained within the 'items' key

We can see this more clearly by wrting the response into a file or importing the json library and using the dumps() method.

Now we can index nested json within response['items'] and extract values from nested key. **Important note len() to see the length of the result.
By default it is 5, add the maxResults parameter to list(), the max value is 50 

import json
from googleapiclient.discovery import build
api_key = 'AIzaSyDoVmGn11RbYZEIaFIuvCpLumzBeyvDmWM'

youtube = build('youtube','v3', developerKey=api_key)

request = youtube.search().list(
    maxResults=25,
    q='avengers',
    part='snippet',
    type='video'
)
response = request.execute()
with open('json_file.json','w') as f:
    for item in response['items']:
        json_str = json.dumps(item, indent=2)
        f.write(json_str)
print(len(response['items']))

The example above uses the type video method and searchs for 25 videos containing title that include avengers
